# Product

## Register

product

## Users
PC gamers playing Growtopia who want to automate repetitive fishing. Used solo, typically minimized or windowed alongside the game. Users are familiar with Growtopia's in-game UI language.

## Product Purpose
AutoFisher automates Growtopia's fishing minigame: the user configures screen positions and image regions for game elements once, then the bot handles bait, water, detonation, and recycling automatically. Success = fish caught without attention.


## Brand Personality
Game-native, playful, reliable. Feels like a tool the Growtopia devs might have shipped inside the game itself.

## Anti-references
- GitHub Dark (current look — too developer/utilitarian, no game personality)
- Generic dark dashboards (shadcn/Radix defaults, Linear, Vercel)
- Windows-style gray system dialogs

## Design Principles
1. Speak the game's language — players should feel immediately at home
2. State is everything — running/paused/stopped must be unmistakable at a glance
3. Configuration is a chore — make setup rows scannable and fast
4. Tool disappears into the task — once fishing runs, the UI fades into the background

## Accessibility & Inclusion
WCAG AA contrast minimum. Pixel font used for headings/labels; readable size enforced. Reduced motion respected.
